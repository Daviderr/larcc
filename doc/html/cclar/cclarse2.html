Importing the library First of all, a modeling application having to deal with simplicial complexes must import the SimpleXn library, denoted smplxn in python. The name of the library was firstly used by our CAD Lab at University of Rome “La Sapienza” in years 1987/88 when we started working with dimension-independent simplicial complexes [PBCF93]. This one in turn imports some functions from the scipy package and the geometric library pyplasm [].
