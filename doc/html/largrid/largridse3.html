3 Cuboidal grids

More interesting is the generation of hyper-cubical grids of intrinsic dimension d embedded in n-dimensional space, via the Cartesian product of d 1-complexes and (n-d) 0-complexes. When d = n the resulting grid is said solid; when d = 0 the output grid is 0-dimensional, and corresponds to a grid-arrangement of a discrete set of points in En.
