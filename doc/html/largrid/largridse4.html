4 Cartesian product of cellular complexes

LAR model of cellular complexes The external representation of a LAR model (necessarily geometrical, i.e. embedded in some En, in order to be possible to draw it) is a pair (geometry,topology), where geometry is the list of coordinates of vertices, i.e. a two-dimensional array of numbers, where vertices are given by row, and topology is a list of cells of fixed dimension d. When d = n the model is solid; otherwise the model is some emberdded d-skeleton (0 ≤ d < n).

Binary product of cellular complexes The larModelProduct function takes as input a pair of LAR models and returns the model of their Cartesian product. Since this is a pair (geometry, topology), its second element returns the topological product of the input topologies.
